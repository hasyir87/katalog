# **App Name**: ScentVault

## Core Features:

- Public Catalog View: Allow public, unauthenticated viewing of perfume data: Number, Nama Parfum, Deskripsi Parfum, Top Notes, Middle Notes, Base Notes, Penggunaan, Sex, Lokasi, Jenis Aroma, Kualitas.
- Authenticated Data Management: Require user authentication to add, edit, or delete perfume data entries.
- AI Perfume Recommendation: Provide perfume recommendations based on data within the database, such as similar notes or use cases. Use existing catalog data to guide recommendations; utilizes a tool to determine relevancy to the query.
- QR Code Generation: Generate a QR code for each 'Jenis Aroma' (type of scent) that, when scanned, leads to a webpage or more info specific to that scent type.
- Database Integration: Persist perfume data using an external SQL database.

## Style Guidelines:

- Primary color: Deep purple (#673AB7), evocative of luxurious scents and exclusivity.
- Background color: Light lavender (#EDE7F6) creates a soft, clean backdrop.
- Accent color: Mauve (#BA68C8) to highlight key elements and calls to action.
- Headline Font: 'Playfair', a modern serif to convey elegance. Body Font: 'PT Sans', for readability in longer text descriptions.
- Use minimalist line icons to represent perfume attributes (e.g., floral, woody, citrus)."
- Employ a card-based layout, similar to the reference, with clear separation of perfume details.
- Implement subtle fade-in animations when new perfume data is loaded or filtered.
